Online Tiffin Booking System 

Overview

This project is a complete and responsive Online Tiffin Booking System designed to connect Customers, Suppliers, and Admin on one platform.
The system is built using React, Node.js, Express API, and MongoDB, offering a smooth user experience with secure and scalable backend operations.

Technologies Used

Frontend

React.js – Component-based UI system for fast and dynamic pages.
Tailwind CSS – Utility-first styling for responsive and modern design.

Backend
Node.js – Server-side JavaScript environment.
Express.js API – Handles routing, authentication, and backend logic.

Database
MongoDB – NoSQL database used to store customers, suppliers, menu items, orders, payments, and feedback.
